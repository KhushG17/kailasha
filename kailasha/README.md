## Local Development

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

Production build:

```bash
npm run build
```

On Windows, if PowerShell blocks `npm`, use:

```bash
npm.cmd run build
npm.cmd run start
```
